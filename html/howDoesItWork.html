How Does it Work?

  1. Chromium rendering engine, Node runtime
  2. All Node.js APIs are available in Electron (e.g. accessing local file system)
  3. The main JS file is run by the main process (opens windows, handles system events)
  4. Each new window runs on its own renderer processes
  5. These communicate with the main process by sending serialised JSON objects via IPC (inter-process communication)
